# **App Name**: GudangCheckout

## Core Features:

- Item Selection: Search and select items for checkout by code.
- Customer Recording: Record customer details for each checkout transaction.
- Quantity Input: Input the quantity of each item being checked out.
- Checkout History: View a comprehensive history of all checkout transactions, filtered by date and customer.
- Checkout logging: Log successful checkouts
- Auto Data Ingestion: AI-powered tool to scan code transaksi or kode resi to pull data from any source

## Style Guidelines:

- Primary color: Emerald green (#50C878) to represent efficiency and reliability.
- Background color: Off-white (#F0F8FF) to provide a clean and professional look.
- Accent color: Forest green (#228B22) for interactive elements and highlights, offering a darker contrast against the lighter green.
- Body and headline font: 'Inter' (sans-serif) for a modern and easily readable interface.
